1. step - add native function definitions in java file e.g. YourJNI.java
2. step - compile .class file
	- in terminal cd class folder
	- javac YourJNI.java
3. step - create header (.h) file
	- go back to parent directory ( ... espeak-jedit/src )
	- then use javah command to create c/c++ header file
	- javah -jni eSpeakServices.ESpeakService
4. step - create native library
